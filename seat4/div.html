Use external style to apply the global formatting of embedded style to multiple pages. External style allows you to specify formats and apply yhem to multiple Web pages rather than just one. External style also is knows as linked style beacuse, instead of listing style specification at the top of your Web page, you create a link to an external document that containts the style code, known as external style sheet. THis method allows you to format a set of Web pages, usch as a Web publication, with a unifrom style and allows you to change the style of all pages later simply by editing the external style sheet. External style can be used in a page together with both inline and embedded styles.
